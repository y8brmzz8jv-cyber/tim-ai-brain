# Dantas Properties

Dantas Properties builds and develops its own commercial properties, then leases them out. Current portfolio (~$12M in assets) is mostly industrial warehouse and flex space. Actively developing a 25-acre industrial park with 100,000+ sq ft of rental space.

**"The Work District"** is the actual brand name for the industrial park portfolio (found in the Dantas Properties Google Drive — a "The Work District Brand" folder exists). Properties are numbered WD 1.0 through WD 6.0+, all on Carroll Ave / Clover Place:

- **WD 1.0** — 3135 Carroll Ave — 2 metal buildings (6K & 7.2K sq ft) — complete since June 2024
- **WD 2.0** — 3200 Carroll Ave — new construction, 9 units — Enterprise Zone Grant eligible. Tenants include **Secret Pickle** (kitchen/restaurant concept) and a court/lounge space. **This is the same project as the [[03 Projects/Veritas Building Company/06 Meetings/2026-06-24 - Court and Tenant Fit-Out Coordination]] meeting note** — Veritas is doing the construction management on it.
- **WD 3.0** — 2713 Clover Place — new construction, 3 units — Enterprise Zone Grant eligible. Tenants include **Legacy** (Legacy Center) and **Delta Star**.
- **WD 4.0** — 2732 Clover Place — new construction, ~30K SF building for **BWXT** (the same BWXT that shows up as a Mr. Pave paving customer — they're also a Work District tenant).
- **WD 5.0** — 2975 Carroll Ave — 26 acres behind Mr. Pave — this is **the 25-acre industrial park** already tracked as the prime directive below, plus an RV Park concept. Matches [[03 Projects/Veritas Building Company/06 Meetings/2026-06-02 - Veritas Weekly Meeting]]'s "Project 5.0 (RV Park)."
- **WD 6.0** — 2941 Carroll Ave — remodeled building, renovate & rent — this was **Traffigo's** building (the business Tim sold — see GOALS.md for the sale terms).

**Enterprise Zone Grant program:** WD 2.0 and WD 3.0 qualify — requires exceeding $100,000 in Qualified Real Property Investments (QRPI) for rehab/expansion.

The Google Drive tracker for all of this ("2025 DESIGN & DEVELOPMENT PROJECTS" spreadsheet, updated weekly) also covers **Veritas Building Company's** full client project roster — see [[03 Projects/Veritas Building Company/CLAUDE.md]] for the current active/potential/closed project list. It's a shared tracker across both companies, not Dantas-only. Left in Drive as the live working document; not duplicated into the vault.

3. Always have the next acquisition/development target identified before the current one wraps

There's no fixed dollar target or timeline on this project — just continued growth. Good candidate for a future Chess Moves session if Tim ever wants to pin down a number.

## Process

**Development SOP** (see `08 System/Development SOP.md` — this is a living, in-progress document, not final):

1. Identify the property
2. Move through the site plan
3. Get permits and approvals with the city
4. Build and develop
5. Get certificate of occupancy
6. Submit for the Enterprise Grant
7. Start marketing
8. Done — fully leased and part of the stabilized portfolio

## Rules & Conventions

- **`(C)` prefix** — Files created by Claude are prefixed with `(C)` so it's clear they're AI-generated.
- **Editing rule** — Before editing any file without the `(C)` prefix, ask for permission first.
- **Skills** — All reusable scripts/automations are saved as markdown files in `09 Skills/`, NOT as Claude Code skills.
- **Meeting notes are always time-stamped.**
- **Every meeting summary must include:** a meeting owner, the decisions made, and the next actions.
- **Cross-company linking:** Any meeting note — in Dantas Properties, Mr. Pave, or Veritas — must link its decision owner back to that person's note in [[00 Notes/People]].
- **The Development SOP is a living document** — update it in place as the process gets clarified with the project manager, don't create a parallel version.
